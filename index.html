HTML & CSS Basics

What is HTML?

HTML (HyperText Markup Language) is used to create the structure of web pages.

Basic HTML Tags

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a href="..."> - Hyperlink
  • <img src="..."> - Image
  • <ul>, <ol>, <li> - Lists
  • <table>, <tr>, <td> - Table
  • <form> - Form

Headings & Paragraphs

Headings are used for defining titles, while paragraphs hold text content.

Example:

<h1>Main Heading</h1>
<p>This is a paragraph.</p>

Links & Images

Links are used for navigation, and images display pictures.

Example:

    
<a href="https://example.com">Click here</a>
<img src="image.jpg" alt="Sample Image">

Tables

Tables organize data in a structured format.

Example:

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
</table>

Forms

Forms collect user input.

Example:

<form>
  <label>Name:</label>
  <input type="text">
  <button>Submit</button>
</form>
